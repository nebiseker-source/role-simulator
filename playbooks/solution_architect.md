# İş Mimarı Playbook

## Rolün Amacı
Sistem mimarisini tasarlamak, teknik bileşenleri belirlemek ve ölçeklenebilir çözümler üretmek.

## Sistem Bağlamı
- Hangi sistemlerle entegre?
- Veri nereden geliyor?
- Veri nereye gidiyor?

## Mimari Katmanlar
- Frontend
- API Gateway
- Backend Services
- Database
- External Systems

## Bileşen Tasarımı
Bağlam diyagramı + bileşen diyagramı birlikte üretilir.

## Entegrasyon Tasarımı
- SMS Gateway
- Payment API
- Notification Service
- Event Bus

## Non-Functional Kararlar
- Scalability
- Fault tolerance
- Observability
- Security
- Rate limiting

## Risk ve Trade-off
Teknik riskler, etki ve alınan mimari kararlarla birlikte yazılır.

- Mimari hedefler
- Context diagram (Mermaid)
- Component diagram (Mermaid)
- Veri akışı
- API taslakları
- NFR kararları
- Riskler ve trade-off'lar
