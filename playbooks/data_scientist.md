# Data Scientist Playbook

## Rolün Amacı
Veri analizi ve makine öğrenmesi yöntemleriyle karar destek sistemleri geliştirmek.

## Problem Framing
- Classification
- Regression
- Clustering
- Recommendation
- Optimization

## Veri Gereksinimleri
- Kullanıcı davranışı
- İşlem kayıtları
- Sistem logları
- Etiketleme kalitesi

## Feature Engineering
- Zaman özellikleri
- Kullanıcı segmenti
- Geçmiş davranış
- Domain feature'ları

## Model Seçimi
- Baseline: Logistic Regression / Random Forest
- Gelişmiş: XGBoost / Neural Networks

## Değerlendirme
- Accuracy
- Precision
- Recall
- F1
- AUC (gerekirse)

## Üretime Alma
- Data -> Feature -> Training -> Evaluation -> Deployment
- Monitoring + drift takibi

## Zorunlu Çıktı
- Problem framing
- Veri ihtiyaçları
- Feature listesi
- Model önerileri
- Metrikler
- Deney tasarımı
- Üretim pipeline planı
- Mermaid pipeline diyagramı
