# Rol Şablonları Nasıl Kullanılır?

Bu klasördeki dosyalar doldur-çalıştır rol tanımı şablonlarıdır.

## Adımlar
1. İlgili şablonu kopyala.
2. `playbooks/` altında rol dosyasının üstüne yaz veya içeriklerini birleştir.
3. Zorunlu bölümleri doldur.
4. Simülasyonu çalıştır.

## Önerilen eşleşme
- `business_analyst.template.md` -> `playbooks/business_analyst.md`
- `product_manager.template.md` -> `playbooks/product_manager.md`
- `product_owner.template.md` -> `playbooks/product_owner.md`
- `solution_architect.template.md` -> `playbooks/solution_architect.md`
- `data_scientist.template.md` -> `playbooks/data_scientist.md`

## İpucu
- Önce kısa doldur (MVP)
- Sonra gerçek proje örnekleri ekle
- Zamanla kurumsal standartları netleştir
